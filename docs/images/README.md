# docs/images — 스크린샷/GIF 넣는 곳

README.md 의 "미리보기" 표가 아래 파일들을 참조합니다. 여기에 캡처를 넣으면 자동으로 표시됩니다.

| 파일명 | 캡처 내용 |
|---|---|
| `scroll.gif` | 스크롤 목록을 좌우로 넘기고, 아이템을 위로 뽑는 장면 |
| `place.gif` | 슬롯/구역에 놓아 붙고 wrap 연출이 재생되는 장면 |
| `roll.gif` | 본 방식으로 스티커가 선 방향으로 돌돌 말렸다 풀리는 장면 |

## 캡처 팁
- **GIF**: Unity Game 뷰를 [ScreenToGif](https://www.screentogif.com) / [LICEcap](https://www.cockos.com/licecap/) / macOS 화면 기록으로 녹화 → GIF 변환
- 권장 크기: 가로 400~600px, 3~5초, 10~15fps
- 본 롤의 입체감은 **Perspective 카메라**나 살짝 기울인 카메라에서 더 잘 보입니다
- 정지 이미지면 `.png` 로 넣고 README 의 확장자만 맞춰 바꾸세요
